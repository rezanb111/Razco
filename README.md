# 🏢 سامانه مدیریت مرخصی شرکت رازکو (Razco)

این یک سیستم هوشمند و شکیل برای مدیریت درخواست‌های مرخصی کارکنان شرکت **رازکو** است.

### 🚀 قابلیت‌ها:
- **پنل مدیریت:** تایید/رد درخواست‌ها و مدیریت کاربران.
- **پنل کارمند:** مشاهده باقی‌مانده مرخصی و ثبت درخواست جدید.
- **داشبورد آماری:** نمایش نمودارهای تحلیلی با Plotly.
- **امنیت:** هش کردن پسوردها و تفکیک سطح دسترسی.

### 🛠 راه اندازی سریع:
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
